Question

What is error code 0060 on MyCarelink Smart (MCL-S) Monitor?

 

Response

At the start of the MCL-S app, the operating system on the mobile device compatibility check failed and no MCL-S app update was available in the App Store.

 

Details

Verify if patient’s mobile device meets requirements (refer to chart by clicking on link under Related Links below). Does caller have alternative device/tablet to use?

  • If yes, assist caller with set up on alternative device.
  • If no, screen patient for alternative monitoring solutions. (Refer to Orders Department for assistance with ordering.)
