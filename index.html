Question

What is error code 0056 on MyCareLink Smart (MCL-S) Monitor?

 

Response

At the start of the MCL-S app, the integrity validation of the MCL-S app check failed.

 

Details

1. Is mobile device/tablet Operating System up to date?

  • If yes, uninstall and reinstall the MCL-S app, then continue to the step 2 below.
  • If no, refer caller to their phone carrier for instruction on how to update OS and to call back once device is up-to-date.

2. Is the Error Code cleared?

  • If yes, walk caller through successful transmission.
  • If no, continue to step 3 below.

3. Does caller have alternative device/tablet to use?

  • If yes, assist caller with set up on alternative device.
  • If no, screen patient for alternative monitoring solutions. (Refer to Orders Department for assistance with ordering.)
