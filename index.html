Question

What is error code 2316 on MyCareLink Smart (MCL-S) Monitor?

 

Response

Unexpected error occurred at any time during the interrogation.

 

Details

Turn off and turn on the Bluetooth on the mobile device.

Remove the reader paired information in Bluetooth settings.

Remove and reinstall batteries.

Start the MCL-S application.

Is the error code cleared?

  • If yes, walk caller through successful transmission.
  • If no, repeat previous steps, switch off and on power for device, unpair all Bluetooth devices followed by turning Bluetooth on\off (make sure to do both and exactly in that order), then walk caller through successful transmission.
