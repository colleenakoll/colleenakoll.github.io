Question

How is the CareLink Monitor (CLM) and information secured on the CareLink Network?

 

Response

1. The Remote Monitor - The data is encrypted and sent over a secure telephone line to a server in our data center.

 

2. The Data Center - This is a state of the art facility with very tight controls in place to control access to the physical equipment.

3. The Network - We have multiple firewalls in place to protect access from external sources, and monitoring of the firewalls to alert us if any intrusions are detected.

 

4. The Servers - We have designed the CareLink application such that it is segmented and we control access to each segment based on specific requirements. This layered method of access and security ensures only the processes or people who need access will get access.

 

5. CareLink home monitors cannot reprogram any therapy parameter in implanted devices (i.e. no remote programming capability).

 

6. For clinicians to access patient information on the CareLink Network, they must login with a username and password, which are used to set up a " Secure Sockets Layer (SSL)" session. (SSL is the standard Internet security protocol.)
