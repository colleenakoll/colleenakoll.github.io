Question

What is error code 3253 on MyCareLink Smart (MCL-S) Monitor?

 

Response

Reader batteries low animation. This animated screen will be seen when the current battery charge level is insufficient to complete an interrogation.

 

Details

Replace the batteries in the Reader with three AA sized alkaline batteries (LR-6 sized). Continue to walk caller through successful transmission. Is the error code cleared?

1. If yes, it is recommended to remove batteries after each use. (May cause corrosion if left in the battery compartment, which makes the Reader non-functional)

2. If no, replace MCL Smart reader.
