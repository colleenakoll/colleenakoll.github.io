Question

Can the Wire T accessory be used on a secure network such as those found at a hospital or a workplace?

 

Response

We have had success by providing the following security settings to the Hospital or workplace Information Technology (IT) Security Administrator.

 

Details

Here is the technical information on Wire T accessory that IT would need to allow:

  • The Wire T needs to connect to any Ethernet port that is allowed outbound access to the internet.
  • The Wire T will make a Dynamic Host Configuration Protocol (DHCP) request on the local network for an IP address and Domain Name System (DNS) servers.
  • The Wire T device will connect to this internet Uniform Resource Locator (URL) backend1.e-device.net (Internet Protocol (IP) address 212.51.178.115) using Secure Sockets Layer (SSL) Transport Layer Security (TLSv1.2).
  • The Wire T must be allow outbound on Transmission Control Protocol (TCP) ports 443 through the firewalls.
